What is Semantic HTML?

Semantic HTML uses elements with meaningful names to describe the structure and content of a webpage. Examples include <article>, <section>, and <header>.

Why Use Semantic HTML?

It improves accessibility, search engine optimization (SEO), and makes the code more readable for developers.
